\doxysubsection{Detailed Description}
Functions to load and free data from the MNIST dataset. 

This contains the implementations for the functions used to read and free data from the MNIST dataset in CSV format. It is used by the neural network in {\ttfamily \doxylink{mnist_8c}{mnist.\+c}}.
