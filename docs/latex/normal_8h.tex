Returns a normally distributed random value with mean mu and standard deviation sigma. 

This function uses an algortihm called \href{https://en.wikipedia.org/wiki/Box\%E2\%80\%93Muller_transform}{\texttt{ Box-\/\+Muller transform}}. Use {\ttfamily normal(0, 1)} to generate a standard normal random variable.
